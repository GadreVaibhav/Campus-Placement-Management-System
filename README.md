# 🎓 Campus Placement Management System

A full-stack web application designed to streamline the campus recruitment process by connecting Students, Recruiters, and Administrators on a single platform. The system automates student registration, placement drives, job applications, interview scheduling, offer management, and recruitment analytics.

## 🚀 Features

### 👨‍🎓 Student Portal
- **Student Registration & Login** (JWT Authentication)
- Complete Profile Management
- Resume Upload & Management
- Browse Available Jobs
- Apply for Placement Drives
- Track Application Status
- View Interview Schedule
- Accept / Reject Placement Offers

### 🏢 Recruiter Portal
- **Recruiter Registration & Login**
- Company Profile Management
- Post New Jobs
- Create Placement Drives
- View Student Applications
- Schedule Interviews
- Generate Placement Offers
- Recruitment Dashboard

### 👨‍💼 Admin Portal
- Secure Admin Dashboard
- Manage Students, Recruiters, and Companies
- Manage Placement Drives and Applications
- Dashboard Analytics
- Student Branch Statistics
- Placement Reports

### 📊 Dashboard Features
- **Key Metrics:** Total Students, Companies, Recruiters, Placement Drives, Applications, Selected Students, and Documents Uploaded.
- **Visualizations:** Students by Branch Chart.
- **Quick Views:** Recent Students, Recent Applications, Upcoming Placement Drives.

## 📖 How to Use the Project
 👨‍🎓 Student
    Open the live application.
    Click Register.
    Select Student as the role.
    Complete your profile.
    Upload your resume.
    Browse available jobs.
    Apply for placement drives.
    Track application status.
    View interview schedules.
    Accept or reject placement offers.
    
 🏢 Recruiter
    Register as a Recruiter.
    Log in to the recruiter dashboard.
    Complete your company profile.
    Post new job openings.
    Create placement drives.
    View student applications.
    Shortlist candidates.
    Schedule interviews.
    Send placement offers.
    
 👨‍💼 Administrator
    Login using Admin credentials.
    Access the Admin Dashboard.
    Manage students.
    Manage recruiters.
    Manage companies.
    Manage placement drives.
    Review all job applications.
    Update application status.
    Monitor placement statistics.
    View analytics and reports.
